Текст на слайде: 
- Язык: Только чистая Java (JDK 11+), без сторонних фреймворков
- Структуры данных: Использование стандартных коллекций (List, Map)

Слайд 4: Архитектура проекта
Заголовок: Распределение модулей (ООП)
Текст на слайде: 
- Модели данных (Model): Классы User, Account, Transaction (перечисление TransactionType)
- Бизнес-логика (Service): AtmService (обработка логина, снятия, переводов)
- Хранение данных (I/O): FileStorage (чтение и запись данных)
- Интерфейс: Main (меню через Scanner)
Оформление: Простая блок-схема: Main -> AtmService -> (Models & FileStorage)

Слайд 5: Основной функционал
Заголовок: Реализованные возможности
Текст на слайде: 
1. Регистрация и авторизация по номеру карты и PIN-коду
2. Блокировка аккаунта после 3 неверных попыток ввода PIN
3. Просмотр баланса и истории транзакций (до 10 последних операций)
4. Пополнение счета и снятие наличных
5. Переводы между пользователями внутри системы
Оформление: Иконки замка, карточки, шестеренок напротив каждого пункта.
